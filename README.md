## 博客
这是一款前后端分离的web应用 —— 博客系统。

**主要功能：**
- `JWT`登录认证
- 博客系统

**技术支持：**
- 前端基于 `vue` 框架开发，部分使用 `element ui`
- 后端基于 `springboot` + `mybatis plus` + `mysql` 开发
- 前后端通信遵循 `RESTful` 风格
